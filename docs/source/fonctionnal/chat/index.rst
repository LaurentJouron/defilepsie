====
Chat
====

Créer une section où les membres peuvent interagir (blog ou tchat). Les utilisateurs pourraient partager leurs défis, 
leurs expériences, poser des questions, etc.
Implémenter des fonctionnalités de messagerie instantanée pour faciliter la communication en temps réel entre les membres.
Mettre en place des fonctionnalités de gestion des droits d'accès pour contrôler qui peut accéder et interagir avec le blog/chat. 
Les personnes épileptique, les accompagnants et les parents pourraient avoir des niveaux d'accès différents.
