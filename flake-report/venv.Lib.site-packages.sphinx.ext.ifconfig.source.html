1 """Provides the ``ifconfig`` directive.
2  
3 The ``ifconfig`` directive enables writing documentation
4 that is included depending on configuration variables.
5  
6 Usage::
7  
8 .. ifconfig:: releaselevel in ('alpha', 'beta', 'rc')
9  
10 This stuff is only included in the built docs for unstable versions.
11  
12 The argument for ``ifconfig`` is a plain Python expression, evaluated in the
13 namespace of the project configuration (that is, all variables from
14 ``conf.py`` are available.)
15 """
